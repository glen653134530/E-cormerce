# ChinaFormationBot

## 1️⃣ Installation locale
1. Installe Python 3.x
2. Installe les dépendances :
   pip install -r requirements.txt
3. Lance le bot :
   python bot.py

## 2️⃣ Hébergement gratuit sur Replit
1. Crée un compte sur https://replit.com/
2. Crée un nouveau projet Python et importe ces fichiers
3. Ajoute ton TOKEN dans config.py
4. Clique sur "Run" pour lancer le bot
5. Active "Always On" pour qu'il tourne 24/7

## 3️⃣ Hébergement gratuit sur Railway
1. Crée un compte sur https://railway.app/
2. Crée un projet "Python"
3. Uploade tous les fichiers
4. Ajoute la variable TOKEN dans les variables d'environnement
5. Déploie et laisse tourner
